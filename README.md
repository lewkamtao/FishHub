  - 项目地址：https://github.com/lewkamtao/FishHub

#### 后端
  - Go + gin + gorm + mysql
  - 项目地址：https://github.com/lewkamtao/FishHub_Server

## 起步

安装依赖

```bash
yarn install
```

## 运行

运行在本地

```bash
yarn dev
```

你需要运行下面命令进行打包编译

```bash
yarn build
```

Checkout the [deployment documentation](https://v3.nuxtjs.org/docs/deployment).
